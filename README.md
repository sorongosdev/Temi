# temi_arduino
1. RFID
2. DHT 센서(습도 측정)
3. 미세먼지 센서
4. 네오픽셀
  - "yellow" 태그 대기
  - "green" 태그 성공, 순찰중
  - "red" 태그 실패
5. 잠금장치(서보모터)
# temi_humidifier
  습도가 40% 이하면 동작
# temi_airpurifier
  미세먼지가 80 이상(위험 수준)이면 동작
